幻书系统

获取方法

  • 可通过抽奖、Boss掉落、商店购买获取幻书
  • 本页面的数据根据版本变动会有偏差属正常情况

幻书 - 理想乡

  • 沁力消耗 - 每次发动消耗[20]点
  • 伤害计算 - [武器基础攻击x幻书完成度]x3段
  • 冷却时间 - [35-武器强化等级]秒
  • +15特效 - 范围扩大50%

幻书 - 无限杀戮

  • 沁力消耗 - 每次发动消耗[2]点
  • 触发概率 - 攻击时[武器强化等级x幻书完成度]%触发
  • 触发效果 - 吸收[5x幻书完成度]点生命
  • +15特效 - 触发时同时吸收精神和饥饿

幻书 - 白银之书

  • 沁力消耗 - 不消耗沁力
  • 特效属性 - 移动速度提升[3x武器强化等级x幻书完成度]%
  • +15特效 - 无

幻书 - 月影之书

  • 沁力消耗 - 每次发动消耗[2]点
  • 触发概率 - 攻击时[武器强化等级x幻书完成度]%触发
  • 触发效果 - 冰冻攻击目标
  • +15特效 - 同时冰冻10范围内所有目标

幻书 - 打工人之书

  • 沁力消耗 - 不消耗沁力
  • 特效属性 - 使武器可进行挖砍铲的工作
  • 特效属性 - 根据武器强化等级和幻书完成度提高效率
  • +15特效 - 无

幻书 - 银泉之书

  • 沁力消耗 - 不消耗沁力
  • 特效效果 - 每隔[120-武器强化等级x5]秒恢复沁力
  • 特效属性 - 恢复沁力3%x幻书完成度
  • +15特效 - 恢复沁力提升[3]%->[5]%

幻书 - 艾雷乌西斯的祭礼之书

  • 沁力消耗 - 不消耗沁力
  • 特效属性 - 作祟武器复活[10]范围玩家
  • 冷却时间 - [2400-100武器强化等级幻书完成度]秒
  • +15特效 - 复活后状态全满

幻书 - 间隙之书

  • 沁力消耗 - 每次发动消耗[5]点
  • 特效效果 - 攻击距离提升[武器强化等级x0.6x幻书完成度]
  • 特效属性 - 攻击时会瞬间移动至目标面前[目标在海上不瞬移]
  • +15特效 - 无

幻书 - 神的雷槌

  • 沁力消耗 - 每次发动消耗[5]点
  • 触发概率 - 攻击时[武器强化等级x幻书完成度]%触发
  • 触发效果 - [3]秒内伤害提升50%,对潮湿目标伤害更高
  • +15特效 - 持续时间提升至[7]秒

幻书 - 万物的血统书

  • 沁力消耗 - 不消耗沁力
  • 特效效果 - 根据当前损失血量提升基础攻击力
  • 特效属性 - 增加[武器基础攻击x当前损失生命百分比x幻书完成度]基础攻击
  • +15特效 - 无

幻书 - 无窗之馆的死者祭宴之书

  • 沁力消耗 - 每次发动消耗[5]点并消耗[3]%生命
  • 触发概率 - 攻击时[武器强化等级x幻书完成度]%触发
  • 触发效果 - 减少目标[2%]最大生命[目标生命30%以下不发动]
  • +15特效 - 消耗生命减少[3]%->[1]%

幻书 - 深绿之书

  • 沁力消耗 - 每收获一个消耗[0.5]点
  • 特效效果 - 收获[武器强化等级]范围内的作物、蜂箱等
  • +15特效 - 可以收获农田

幻书 - 至高调理冥想书

  • 沁力消耗 - 不消耗沁力
  • 触发概率 - [武器强化等级x幻书完成度]%
  • 触发效果 - 使用烹饪锅时有一定的概率获得双倍收获
  • +15特效 - 无

幻书 - 实相仪经

  • 沁力消耗 - 每次发动消耗[10]点
  • 触发概率 - 攻击时[武器强化等级x幻书完成度]%触发
  • 触发效果 - 额外造成[武器基础攻击x幻书完成度/2]的真实伤害
  • +15特效 - 额外伤害提升[50]%->[100]%

幻书 - 荒暴狂乱的杀剧之宴

  • 沁力消耗 - 不消耗沁力
  • 特效效果 - 连续进行攻击积累杀意值达到各阶段触发多种效果
  • 特效效果 - 每次攻击时附加[杀意值/10]点真实伤害
  • 注意事项 - [5x幻书完成度]秒内无攻击行为将会清空杀意值
  • +15特效 - 附加伤害提升[杀意值/10]->[杀意值/5]

幻书 - 森罗万象的统御者

  • 沁力消耗 - 不消耗沁力
  • 特效效果 - 减少所有沁力消耗[4x武器强化等级x幻书完成度]%
  • +15特效 - 无
